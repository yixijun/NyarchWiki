    Installation

    如果您已经安装了 Linux 发行版，Nyarch Linux 的安装过程是相同的。

    如果您还没有安装过 Linux 发行版，请考虑安装另一个发行版。 如果您仍然想继续，那么您可以决定在以下位置安装 Nyarch：

    1. 您的计算机
    2. 虚拟机
    3. 在现有的 Linux 发行版上

    在您的计算机上安装 Nyarch

    前置条件

    • 一台符合资源要求的计算机
    • 至少 4GB 容量的 USB 闪存盘
    • 已经有一台可以工作的计算机（或者如果您使用 Etchdroid，则可以使用 Android 设备）
    • 下载了正确的 Nyarch ISO 文件。ISO 文件可以在 https://nyarchlinux.moe/ 找到。如果您有 Nvidia GPU，请在这里检查支持
    • 备份您计算机上的每个重要文件

    创建可启动驱动器

    1. 下载 Balena Etcher 或任何类似的程序（如果您更熟悉它们）
    2. 将 USB 驱动器插入您的计算机
    3. 打开 Balena Etcher

    然后点击"Flash from file"。 4. 选择您之前下载的 Nyarch Linux ISO 文件。 5. 点击 select Target

    然后选择您的 USB 驱动器 注意：确保选择正确的驱动器，因为您将丢失其上的任何文件。 6. 点击"Flash"并等待完成。

    选择安装方法

    您可以决定以以下方式安装 Nyarch： 1. 删除整个磁盘并安装 Nyarch 2. 在同一磁盘上与 Windows 一起安装 Nyarch 3. 在另一个磁盘上与 Windows 一起安装 Nyarch 如果您想执行 1 或 3，则跳到下一部分。 如果您想执行 2，则必须缩小您的 Windows 分区。

    缩小 Windows 分区
    1. 启动 Windows
    2. 哭泣
    3. 启动在"创建和格式化硬盘分区"下找到的 磁盘管理 实用程序。
    4. 右键单击主 Windows 分区并选择"压缩卷..."
    5. 将分区大小缩小至少 30GB。这将是留给 Linux 的空间

    启动 Live USB

    1. 将 USB 驱动器连接到您的设备
    2. 访问您设备的 BIOS/UEFI 实用程序。访问的过程取决于您的设备。 如果您无法弄清楚如何访问 BIOS，请在设备启动时疯狂按 F 键，同时喊 Hora Hora Hora，直到找到正确的键。
    3. 禁用安全启动
    4. 找到启动顺序的设置，并将您的 USB 驱动器放在第一位。或者，如果您可以访问启动菜单，请从那里启动它。
    5. 保存并退出
    6. 希望 Nyarch Live 能够启动。如果没有，请检查您是否确实将 USB 设置在第一位，然后在 Nyarch 支持频道（或任何 Linux 社区支持）中寻求帮助。

    Nyarch Live 环境

    您希望已成功启动 Nyarch。 现在您处于 Live 环境中。

    当您安装 Linux 发行版时，live 环境就像"先试后买"的体验。 它是一个临时操作系统，您可以从 USB 驱动器或 CD 运行，而无需实际将其安装在计算机的硬盘驱动器上。

    这样，您可以测试 Linux 发行版的工作方式，特别是 Wifi、硬件设备、声音等...。您还可以探索 Nyarch 的一些功能，但请考虑与已安装的系统相比，性能较差，这是由于 USB 驱动器的 I/O 速度。

    如果您在 live 环境中需要执行需要输入用户密码的操作，密码是 live。

    Nyarch 安装程序

    安装程序将在系统启动时自动启动。

    然后它会提示您输入本地化设置。根据您的偏好填写它们。

    分区

    注意：在这部分要小心，因为错误可能会导致数据丢失！

    在这里您可以选择多种内容： - 是否擦除整个磁盘并在其上安装 Nyarch。您可以在窗口的上部选择不同的磁盘。

    • 是否用 Nyarch 安装替换分区（图像中未显示）

    • 是否与另一个操作系统一起安装 Nyarch（图像中未显示）

    • 手动选择您的分区（如果您知道自己在做什么）

    您还可以选择：

    • Swap：它就像存储在磁盘上的额外 RAM。强烈建议启用它。

    • 文件系统：您可以在以下文件系统之间进行选择

    • ext4：是 Linux 发行版的标准。在许多情况下它更快，并且被认为更稳定。

    • btrfs：支持即时快照，对数据损坏和冗余有更多保护。在某些情况下稍慢。

    • xfs：除非您知道自己在做什么，否则不要安装它

    完成安装

    选择您的名称，等待安装完成，然后重新启动！

    在虚拟机上安装 Nyarch

    • 建议用于安装 Nyarch 的 VM 软件：VMWare Workstation、QEMU、Gnome Boxes

    • 不建议用于安装 Nyarch 的 VM 软件：VirtualBox（因为 3D 加速不好）

    Nyarch 大量使用动画来增强用户体验，因此在 VM 中启用 3D 加速。否则它会卡得要命。兄弟，你不想看到比 Blue Lock S2 更糟糕的动画，相信我。 创建具有以下规格的 VM：

    • CPU：尽可能多/根据您的喜好

    • 内存：至少 3GB，建议 6GB 以获得流畅的系统

    • 存储：安装操作系统至少需要 30GB

    • 支持 UEFI/BIOS

    创建并启动 VM 后，跳到 Nyarch Live 环境
